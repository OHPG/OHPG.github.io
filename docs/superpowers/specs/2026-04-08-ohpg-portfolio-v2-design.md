## 1. Concept & Vision

一个沉浸式的全屏翻页作品集，采用玻璃拟态（Glassmorphism）设计语言。每屏展示一个项目，配合毛玻璃卡片和柔和渐变背景，通过 3D 视差滚动和炫酷动画打造强烈视觉冲击力。整体风格：现代、酷炫、有深度。

---

## 2. Design Language

### 色彩系统

| Token | Hex | 用途 |
|-------|-----|------|
| `--bg-start` | `#667eea` | 背景渐变起点（蓝紫） |
| `--bg-end` | `#764ba2` | 背景渐变终点（紫红） |
| `--glass-bg` | `rgba(255, 255, 255, 0.15)` | 毛玻璃背景 |
| `--glass-border` | `rgba(255, 255, 255, 0.3)` | 毛玻璃边框 |
| `--glass-shadow` | `rgba(0, 0, 0, 0.2)` | 毛玻璃阴影 |
| `--text-primary` | `#ffffff` | 主文本（白色） |
| `--text-secondary` | `rgba(255, 255, 255, 0.8)` | 次要文本 |
| `--accent` | `#ff6b9d` | 强调色（粉色） |

### 字体

- **主字体**: Outfit（Google Fonts），回退 `-apple-system, BlinkMacSystemFont, sans-serif`
- **字重**: 400 (正文), 500 (中等), 700 (标题), 800 (强调)

### 间距系统

- 基础单位: `8px`
- 常用间距: `8`, `16`, `24`, `32`, `48`, `64`, `96px`

### 动效规范

| 类型 | 时长 | 缓动 | 用途 |
|------|------|------|------|
| 翻页过渡 | `800ms` | `cubic-bezier(0.77, 0, 0.175, 1)` | 全屏切换 |
| 视差滚动 | `实时` | - | 背景/内容分层移动 |
| 毛玻璃出现 | `600ms` | `ease-out` | 卡片入场 |
| 内容淡入 | `400ms` | `ease-out` | 元素依次出现 |
| hover 反馈 | `200ms` | `ease` | 交互反馈 |

### 视觉元素

- **背景**: CSS 多层渐变 + 动态模糊光斑
- **卡片**: 毛玻璃效果 `backdrop-filter: blur(20px) blur()`
- **边框**: 1px 半透明白边 + 微发光
- **阴影**: 多层柔和阴影营造深度
- **图标**: Remixicon 图标库

## 4. Features & Interactions

### 4.1 全屏翻页

- 滚动触发翻页切换（wheel 事件，节流）
- 点击左右区域或分页指示器切换
- 键盘上下箭头支持
- 触摸滑动支持（移动端）
- 翻页动画：当前屏缩小淡出 + 新屏放大淡入

### 4.2 3D 视差滚动

- 背景层：多层渐变以不同速度移动
- 光斑层：模糊光斑随滚动旋转移动
- 内容层：卡片轻微 3D 倾斜

### 4.3 毛玻璃卡片

- `backdrop-filter: blur(20px)`
- 白色 15% 透明背景
- 1px 白色 30% 边框
- 多层阴影营造悬浮感
- 内容入场时依次淡入（stagger 100ms）

### 4.4 分页指示器

- 4 个圆点代表 4 个项目
- 当前项目高亮（填充色 + 放大）
- 点击可跳转
- 切换时有过渡动画

### 4.5 项目内容

- Logo + 名称（顶部）
- 简短描述
- 功能特点（图标 + 文字列表）
- 截图缩略图（可点击查看大图）
- GitHub 和 Release 链接按钮

## 5. Component Inventory

### 5.1 Background

|------|------|

## 6. Technical Approach

### 技术栈

- **纯 HTML + CSS + Vanilla JS**（无框架依赖）
- **CSS**: CSS Variables + Flexbox/Grid + CSS 3D Transforms
- **动画**: CSS Transitions + requestAnimationFrame
- **图标**: Remixicon CDN
- **字体**: Google Fonts Outfit

### 文件结构

```
docs/
└── v2.0/
    └── index.html      # 单文件包含所有 HTML/CSS/JS
```

### 性能优化

- 翻页节流（防止快速滚动触发多次）
- CSS `will-change` 用于动画元素
- 触摸事件 passive: true
- 图片懒加载
