# OHPG 作品集原型设计文档

**日期**: 2026-04-08
**版本**: v1.0
**状态**: 已批准

---

## 1. Concept & Vision

一个科技感十足的深色主题作品集，展示 OHPG 的四个 HarmonyOS + Jellyfin 应用。采用单页滚动设计，首屏以简洁有力的项目卡片吸引注意力，滚动后展开详情，让访客能够快速了解每个应用的核心功能和多端适配能力。整体风格：深蓝黑底 + 霓虹青色渐变，流畅优雅的动画过渡。

---

## 2. Design Language

### 色彩系统

| Token | Hex | 用途 |
|-------|-----|------|
| `--bg-primary` | `#0a0e17` | 主背景（深蓝黑） |
| `--bg-secondary` | `#111827` | 卡片/区块背景 |
| `--bg-tertiary` | `#1e293b` | 悬停/激活态 |
| `--accent-cyan` | `#3bf4fb` | 主强调色（霓虹青） |
| `--accent-gradient` | `linear-gradient(135deg, #3bf4fb 0%, #a855f7 50%, #ec4899 100%)` | 渐变 |
| `--text-primary` | `#f8fafc` | 主文本 |
| `--text-secondary` | `#94a3b8` | 次要文本 |
| `--border-glow` | `rgba(59, 244, 251, 0.3)` | 发光边框 |

### 字体

- **主字体**: Inter（Google Fonts），回退 `-apple-system, BlinkMacSystemFont, sans-serif`
- **字重**: 400 (正文), 500 (中等), 600 (标题), 700 (强调)

### 间距系统

- 基础单位: `8px`
- 常用间距: `8`, `16`, `24`, `32`, `48`, `64`, `96px`
- 容器最大宽度: `1200px`

### 动效规范

| 类型 | 时长 | 缓动 | 用途 |
|------|------|------|------|
| 淡入 | `400ms` | `ease-out` | 元素进入视口 |
| 位移 | `600ms` | `cubic-bezier(0.16, 1, 0.3, 1)` | 卡片/区块滑入 |
| 悬停 | `200ms` | `ease` | 交互反馈 |
| 图片切换 | `300ms` | `ease-in-out` | 设备预览图交叉淡化 |

### 视觉元素

- **背景**: Canvas 粒子动画（稀疏粒子 + 动态连线），减少性能消耗
- **卡片边框**: 1px `rgba(255,255,255,0.1)` + 悬停发光边框
- **图标**: Remixicon 图标库
- **阴影**: `box-shadow: 0 4px 24px rgba(0,0,0,0.4)`

## 4. Features & Interactions

### 4.1 固定导航栏

- 滚动时背景 `backdrop-filter: blur(12px)` 模糊
- 移动端 Hamburger 菜单，全屏 overlay
- 点击项目名平滑滚动到对应区域

### 4.2 Hero 区域

- 标题文字渐入动画（stagger 100ms）
- 项目卡片 hover: `translateY(-4px)` + 发光边框
- 粒子背景随鼠标轻微响应（性能优化：仅移动端启用）

### 4.3 项目详情卡片

**设备切换 Tab**:
- 点击切换设备类型（移动/平板/桌面）
- Tab 有波纹点击反馈
- 内容区图片交叉淡化切换（300ms）

**截图预览**:
- 大图展示当前选中截图
- 缩略图指示器可点击切换
- 支持触摸滑动切换（移动端）

**功能特点**:
- 图标 + 文字列表
- 入场时依次淡入（stagger 80ms）

### 4.4 全局交互

- 滚动触发 AOS 动画（fade-up, slide-up）
- 图片懒加载（IntersectionObserver）
- 平滑滚动 `scroll-behavior: smooth`

## 6. Technical Approach

### 技术栈

- **纯 HTML + CSS + Vanilla JS**（无框架依赖）
- **CSS**: CSS Variables + Flexbox/Grid
- **动画**: CSS Transitions + AOS 库（本地存储于 `assets/css/aos.css`）
- **图标**: Remixicon CDN
- **字体**: Google Fonts Inter

### 文件结构

```
docs/
└── v1.0/
    └── index.html      # 单文件包含所有 HTML/CSS/JS

assets/                 # 外部依赖（CSS/JS/Images）
```

### 性能优化

- 粒子动画：桌面端禁用或减少粒子数量
- 图片懒加载：IntersectionObserver
- CSS `will-change` 用于动画元素
- `passive: true` 用于触摸事件监听
